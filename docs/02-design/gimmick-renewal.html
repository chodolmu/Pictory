Pictory Gimmick Renewal

Color-Flood + Row Destruction Puzzle — Obstacle System Redesign v2

Design Philosophy

Royal Match, Candy Crush, Gardenscapes 등 성공 퍼즐게임의 공통 원칙을 Pictory의 컬러플러드 메커니즘에 맞게 적용합니다.

Renewed Gimmick Catalog (11)

🧊

Ice — 얼음

Durability

Mechanics

  • BFS 리컬러 가능, 파괴 가능
  • 내구도 1~3단계 (스테이지별 설정)
  • 파괴 1회마다 내구도 -1
  • 내구도 0이 되면 얼음 제거
Counter Skill: rainbow_wave — 3x3 일괄 타격으로 내구도 빠르게 깎기

Example Play

2
1
1 빨간색 터치 → 2행 전체 BFS 리컬러
2 2행 라인 완성 → 파괴!
3 얼음(2단) → 1단으로 금 갈림
4 얼음(1단) → 제거됨. 추가 1턴 소모!
🔒

Locked — 잠금

Restrict

Mechanics

  • BFS 그룹에 포함되지만 리컬러 불가
  • 원래 색이 고정됨
  • 라인 파괴로만 제거
  • 잠금칸이 포함된 행에서 색을 맞추기 어려움
Counter Skill: color_swap — 필드 전체 두 색 교환으로 잠금칸 색에 맞추기

Example Play

🔒
1 1행: 파란 4개 + 초록(잠금) 1개
2 파란으로 BFS → 잠금칸은 초록 유지!
3 라인 미완성. 초록을 어떻게 처리할지 고민 필요
4 초록으로 행을 통일해야 파괴 가능
🪨

Stone — 돌

Structure

Mechanics

  • BFS 완전 차단
  • 파괴 불가, 리컬러 불가
  • 중력 무시 (고정)
  • 보드의 영구 벧
Counter Skill: 없음 (영구 장애물) — 우회 전략으로만 대응

Example Play

🪨
1 1행: 빨간 4개 + 돌 1개
2 BFS가 돌을 넘지 못함 → 좌2 / 우2 분리
3 라인 미완성 (돌은 파괴 불가)
4 돌 양쪽을 각각 처리해야 함
🎭

Curtain — 커튼 NEW

Information

Mechanics

  • 셀의 색이 보이지 않음 (? 표시)
  • BFS 리컬러하면 커튼이 벗겨지며 색 공개
  • 리컬러 전까지는 무슨 색인지 모름
  • 계획 수립을 방해하는 정보 차단 기믹

Royal Match의 Curtain에서 착안. 단순하지만 전략적 판단을 크게 방해.

Counter Skill: future_eye (미래의 눈) — 커튼 뒤 색을 미리 확인

Example Play

?
?
1 1행: 빨 ? 빨 ? 빨 — ?가 무슨 색인지 모름
2 빨간으로 BFS → ?칸이 빨간으로 칠해지며 공개
3 만약 원래 파란이었다면? → 빨간으로 바뀌며 커튼 해제
4 운이 좋으면 라인 완성, 나쁘면 턴 낭비
☣️

Taint — 오염 NEW

Active Threat

Mechanics

  • 매 턴 종료 시 인접 1칸을 "오염색"으로 감염
  • 오염색은 어떤 일반 색과도 매치 안 됨
  • BFS 리컬러로 오염 셀을 덮어칠하면 정화
  • 방치하면 증식 — 가장 위험한 능동 장애물

Candy Crush Chocolate + Spread 리워크. 방치 페널티가 핵심.

Counter Skill: color_storm / color_swap — 오염 셀을 일반 색으로 밀어내기

Example Play

☣
T1 오염(1,0) 존재. 다른 곳 플레이.
T2 턴 종료 → 오염이 아래칸(1,1)을 감염!
T3 이제 오염 2칸. 방치하면 3, 4칸으로...
! 파란으로 BFS 리컬러 → 오염칸이 파란으로 정화됨
💣

Bomb — 시한폭탄 NEW

Active Threat

Mechanics

  • N턴 카운트다운 표시 (기본 4턴)
  • 0이 되면 3×3 범위를 랜덤 색으로 섫음
  • 폭탄 셀을 포함한 라인 파괴로 안전 제거
  • BFS 리컬러/파괴 모두 가능

Candy Crush Candy Bomb에서 착안. 우선순위 판단을 강제.

Counter Skill: row_clear — 폭탄 포함 행/열을 긴급 파괴

Example Play

3
T1 폭탄(1,1) 카운트 3
T2 턴 종료 → 카운트 2
T3 턴 종료 → 카운트 1 ⚠️
T4 미처리 시 → 폭발! 주변 3×3 색 혼란
✓ 3턴 안에 라인에 포함시키면 안전 제거
⛓️

Chain — 사슬 NEW

Restrict

Mechanics

  • BFS 리컬러 가능, 하지만 파괴 불가
  • 인접 셀의 파괴(라인 완성)로만 사슬 해제
  • 사슬 해제 후 다음 턴부터 정상 파괴 가능
  • 2단계 처리 필요 → 우회 퍼즐

Royal Match Chain에서 착안. 간접 해제 메카닉.

Counter Skill: color_bomb — 사슬 인접 셀을 폭파하여 해제 유도

Example Play

⛓
1 2행: 빨간 4개 + 사슬(빨간) 1개
2 라인 파괴 시도 → 사슬칸만 파괴 실패!
3 먼저 1행 또는 3행을 파괴해서 인접 파괴 발생
4 사슬 해제! 다음 턴에 정상 파괴 가능
🧱

Wall — 벽 NEW

Structure

Mechanics

  • 셀이 아닌 셀 경계에 배치
  • 해당 방향의 BFS 확산을 차단
  • 인접 파괴 N회로 제거 가능 (내구도 1~2)
  • 돌보다 정밀한 차단 — 공간은 살려두되 연결만 끔음

Royal Match / Gardenscapes Wall에서 착안. 경로 설계 퍼즐.

Counter Skill: row_clear — 행/열 전체 파괴는 벽을 무시하고 관통

Example Play

1 1행: 빨3 | 벽 | 파2 — BFS가 벽을 넘지 못함
2 빨간으로 칠해도 왼쪽 3칸만 빨강. 라인 미완성
3 벽 양쪽을 각각 같은 색으로 만들어야 함
4 인접 라인 파괴로 벽 내구도 깎기 가능
💥

Crack — 균열 NEW

Chain Reaction

Mechanics

  • BFS/파괴 모두 정상 동작
  • 파괴되면 상하좌우에 얼음(내구1) 생성
  • 빨리 처리하면 피해 최소, 방치하면 얼음 확산
  • 얼음 기믹과 시너지 — 균열이 많은 스테이지는 얼음 지옥
Counter Skill: rainbow_wave — 3×3 범위로 균열+생성된 얼음 동시 처리

Example Play

💥
1 2행: 빨간 5개 (균열 포함) → 라인 파괴!
2 균열 파괴 → 상하좌우 4칸에 얼음 생성
3 (0,1) (2,1) 에 얼음 추가 + 아래행에도
! 얼음 제거에 추가 2~4턴 소모
⚓

Anchor — 앵커

Structure

Mechanics

  • BFS/리컬러/파괴 모두 정상
  • 중력 냉하 불가 (위치 고정)
  • 파괴 시 앵커 제거
  • 냉하 패턴을 방해하여 보드 배치 고정
Counter Skill: shuffle — 전체 재배치로 고정 패턴 리셋

Example Play

⚓
1 1행 파괴 → 위에서 새 블록 냉하
2 앵커(0,1)는 고정 — 냉하 안 함
3 앵커 아래 빈칸이 생김 (불규칙 배치)
4 앵커를 파괴해야 정상 냉하 복구
⏳

Fade — 퇴색

Timing

Mechanics

  • BFS 리컬러 후 2턴 뒤 원래 색으로 복귀
  • 리컬러 시 카운터 시작/리셋
  • 원래 색으로 리컬러하면 카운터 비활성
  • 라인 완성 타이밍을 압박하는 기믹
Counter Skill: times_breath (+3턴) — 퇴색 복귀 전에 라인 완성할 여유 확보

Example Play

T1 1행: 파/초/빨/초/파 (퇴색칸 3개)
T1 초록으로 BFS → 퇴색칸도 초록으로 변경
T2 카운터 2 → 1
T3 카운터 0 → 퇴색칸이 원래 색으로 복귀!
! 2턴 안에 라인 완성 못 하면 원점으로

Combo Synergies — 기믹 조합으로 난이도 급상승

단독으로는 쉽지만, 조합하면 기하급수적으로 어려워지는 조합 예시.

💣 Bomb + ☣️ Taint
폭탄 처리하려면 오염 방치, 오염 처리하려면 폭탄 터짐. 우선순위 딪레마.
🔒 Locked + ⏳ Fade
잠금칸은 색 변경 불가 + 퇴색칸은 2턴 후 복귀. 라인 완성 타이밍 극상.
💥 Crack + 🧊 Ice (다층)
균열 파괴 → 얼음 생성 → 기존 얼음과 겨쳐 얼음 지옥.
🎭 Curtain + ⛓️ Chain
커튼 뒤 색 모르는데, 사슬은 인접 파괴로만 해제. 정보 부족 + 우회 필요.
🧱 Wall + ☣️ Taint
벽이 BFS 차단 → 오염 정화용 BFS가 벽 너머로 못 감. 오염 확산 방치.
⚓ Anchor + 💣 Bomb
폭탄이 고정되어 냉하로 자연 제거 불가. 반드시 제자리에서 처리해야.
