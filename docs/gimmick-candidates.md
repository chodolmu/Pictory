# Pictory — 기믹 & 동료 스킬 후보 리스트

> 상태: **v1 확정** (2026-04-04)
>
> 기본 규칙: 기믹의 "턴" 트리거는 **플레이어의 1회 색칠 액션** 기준.
> 스토리 모드에서는 턴 소모, 인피니티 모드에서는 턴 개념 없이 액션 횟수로 트리거.
>
> 그리드 크기: 스테이지별로 가변 (초반 5×5 등 → 최대 7×7). 버퍼 행 수는 그리드 행 수와 동일.
>
> 제작 원칙: **AI-only 구현** — 외부 리소스 없이 코드만으로 구현. 기믹 비주얼은 Godot 내장 기능(도형, 색상, 셰이더, Tween, 파티클)으로 표현.

# Part 1. 맵 기믹 (스테이지에 배치되는 요소)

---

## 공용 기믹 (스토리 + 인피니티 모두 등장)

| # | 기믹 | 설명 | 전략 영향 | 비고 |
|---|------|------|----------|------|
| M1 | **잠긴 칸** | 색 변경 불가. 라인 완성 파괴로만 제거됨 | 라인 완성 루트를 강제함 | |
| M2 | **돌 칸** | 색 없음, BFS 연결 차단, 파괴 불가 | 그리드 형태 자체를 변형 | 스토리: 레벨 디자인 배치 / 인피니티: 특정 구간에서 등장 |
| M3 | **얼음 칸** | 2번 라인 파괴에 포함돼야 제거됨 (1회차에 금 감, 2회차에 파괴) | 같은 라인을 두 번 완성해야 하는 도전 | |
| M4 | **무지개 칸** | 모든 색과 연결 판정 (와일드카드) | 대형 그룹 형성 촉진, 라인 완성 보조 | |
| M5 | **앵커 칸** | 중력 무시, 파괴될 때까지 위치 고정 | 낙하 패턴 변형 | 앵커 아래 공간은 버퍼에서 정상 충전. 앵커는 해당 열의 "천장" 역할 |
| M6 | **페인트통** | 이 칸을 칠하면 같은 행 또는 열 전체가 해당 색으로 변환 | 라인 완성 트리거, 강력한 전략 도구 | 행/열 중 어느 쪽인지는 칸에 표시 |
| M7 | **코인 칸** | 파괴 시 추가 재화 획득 | 선택적 보상 목표 | |
| M8 | **연쇄 칸** | 이 칸 포함 파괴 시 파괴 수 2배 카운트 | 목표 달성 가속 | |

## 스토리 모드 전용 기믹

> 턴 제한이 있는 스토리 모드에서만 의미 있는 기믹.

| # | 기믹 | 설명 | 전략 영향 | 비고 |
|---|------|------|----------|------|
| S1 | **별 칸** | 파괴 시 보너스 턴 +1 | 턴 리소스 관리 | 인피니티에서는 S4 시간 칸이 대응 |
| S2 | **번짐 칸** | 매 턴 종료 시 인접 1칸을 자기 색으로 감염 | 방치하면 라인 완성 계획이 엉킴 | 능동적으로 퍼짐 → 빨리 처리해야 |
| S3 | **퇴색 칸** | 칠해도 2턴 후 원래 색으로 복귀 | 라인 완성 타이밍을 정확히 맞춰야 함 | 시간 압박이 없는 스토리에서 퍼즐성 강화 |

---

## 인피니티 모드 전용 기믹

> 시간 제한 + 속도 중심의 인피니티 모드에서만 의미 있는 기믹.

| # | 기믹 | 설명 | 전략 영향 | 비고 |
|---|------|------|----------|------|
| S4 | **시간 칸** | 파괴 시 보너스 시간 +N초 | 시간 리소스 관리 | 스토리의 별 칸(S1)에 대응 |
| S5 | **독 칸** | 필드에 존재하는 동안 시간 감소 속도 증가 (1.5배 등) | 빨리 제거해야 하는 긴급 목표 | 여러 개 쌓이면 가속 |

# Part 2. 동료 이마젠 스킬 (플레이어가 능동적으로 사용)

> 편성한 동료 이마젠이 보유한 능력.
> 플레이어가 원하는 타이밍에 발동. **턴 미소모**, 쿨타임 있음.
> 스킬은 범용적이어야 하며, 특정 기믹에 종속되지 않음.

---

## 공용 스킬 (스토리 + 인피니티 모두 사용 가능)

| # | 스킬 | 설명 | 용도 |
|---|------|------|------|
| K1 | **단색 폭풍** | 필드 위 지정 색상 블록 N개를 다른 색으로 변환 | 라인 완성 보조, 색 정리 |
| K2 | **무지개 파동** | 지정 영역(3×3) 내 모든 칸을 선택한 색으로 통일 | 강제 라인 완성, 긴급 정리 |
| K3 | **컬러 스왑** | 필드 전체에서 두 색상을 서로 교환 | 보드 전체 구조 변경 |
| K4 | **컬러 봄** | 필드 위 특정 색 블록 N개를 즉시 제거 (중력 적용) | 대규모 정리 + 낙하 유도 |
| K5 | **행 청소** | 지정 행 또는 열 전체를 즉시 파괴 | 직접적 라인 제거 |
| K6 | **되감기** | 직전 1회 액션 상태로 보드 복구 (Undo) | 실수 복구 |
| K7 | **미래의 눈** | 최적 수순 1턴 힌트 표시 | 막혔을 때 가이드 |
| K8 | **셔플** | 필드 전체 블록 위치를 랜덤 재배치 (색 비율 유지) | 막힌 보드 리셋 |
| K9 | **큐 뒤집기** | 버퍼 큐의 색상 순서를 역순으로 뒤집음 | 다음에 떨어질 블록 색을 전략적으로 변경 |
